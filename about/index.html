👋 你好，我是 [Ling]

☕ Java 开发者 | 🤖 AI 智能体探索者 | 🚀 终身学习者

欢迎来到我的博客！我是一名热衷于技术落地的后端开发者。
从构建高可用的微服务架构，到探索 LLM（大语言模型）的智能体应用，我始终走在技术探索的路上。

🛠️ 技术栈 (Tech Stack)

💻 后端核心 (Core Backend)

稳扎稳打的 Java 基础与企业级开发经验。

  • Java: 核心语言，熟悉 JUC 并发编程与 JVM 调优。
  • Spring Ecosystem: 熟练掌握 Spring Boot 快速开发与 Spring Cloud 微服务治理。
  • Redis: 熟悉缓存策略、分布式锁及高并发场景下的应用。
  • ☁️ Microservices: 理解微服务架构设计，熟悉服务注册发现、配置中心与熔断降级。

🤖 AI 与大模型 (AI & LLM)

正在探索将传统后端与 AIGC 能力结合的无限可能。

  • 🦜🔗 LangChain: 熟练使用 LangChain 编排 LLM 逻辑。
  • 🕵️ Agents: 致力于构建自主智能体（Autonomous Agents），实现工具调用与复杂任务规划。
  • 🧠 RAG: 了解检索增强生成技术，搭建私有知识库问答系统。

💡 我在做什么 (What I Do)

我不仅仅是写代码，更关注如何用技术解决实际问题。我的关注点包括：

  1. 后端架构现代化：如何设计高内聚低耦合的微服务系统。
  2. AI 应用落地：如何将 LangChain 智能体集成到 Spring Boot 应用中，让 Java 应用拥有“大脑”。
  3. 效能提升：利用 AI 辅助编码与自动化运维。
